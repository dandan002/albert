  <action>
Implement `albert.backtest` with a `run_backtest(market_id, strategy_class, config)` function:
1. Initialize strategy with provided config.
2. Query `orderbook_snapshots` for given `market_id` ordered by `timestamp`.
3. Loop through snapshots:
   - Convert snapshot to `MarketDataEvent`.
   - Call `strategy.on_market_data(event)`.
   - Record "fills" at snapshot `yes_bid`/`yes_ask` if strategy emits `OrderIntent`.
   - Track cumulative P&L and drawdown.
4. Print summary: Total Return, Win Rate, Max Drawdown.

Include a `main` block for CLI usage: `python -m albert.backtest <market_id> <strategy_path> [config_json]`.
